## 7. DQ Rules

| Check | Layer | Level | Description | Failure condition |
|---|---|---|---|---|
| ts_not_null | normalized | FAIL | Временная метка Время, ГГГГ-ММ-ДД ЧЧ:ММ:СС должна быть заполнена. | Содержит NULL. |
| ts_monotonic | normalized | WARNING | Временная метка должна монотонно возрастать в рамках одной загрузки. | Значения не возрастают (например, из-за ошибки сортировки). |
| temperature_range | normalized | WARNING | Температура должна находиться в разумном физическом диапазоне. | Значение < -80°C или > 60°C. |
| humidity_range | normalized | WARNING | Относительная влажность должна быть в допустимом диапазоне. | Значение < 0% или > 100%. |
| precipitation_non_negative | normalized | FAIL | Количество осадков не может быть отрицательным. | Значение < 0. |
| unique_city_ts | normalized | FAIL | Комбинация города и временной метки должна быть уникальной. | Обнаружены дубликаты по ключу (city_id, ts). |
